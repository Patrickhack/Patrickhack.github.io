Java Project: E-commerce bookstore

E-commerce bookstore website based on Java Spring MVC framework , Hibernate and MySQL. Bootstrap as frontend.

It has two part, Client web and AdminPortal. In Client-side, a client can look around, search and buy book. When clients create an account or place an order, they will receive an email confirmation. In the Server-side we can add or delete books and pop the order that client already placed.

Python Project: Malware Analysis Run Time Prediction

Selected Malware from Static analysis lab and Generated Dynamic analysis report by using Cuckoo.

Used SKLEARN to Compare four algorithms performance: Decision Trees, Random Forests, K Nearest Neighbors, and Support Vector Machines by Generating ROC Curves and Confusion Matrix .

Generating pickles to build a prediction web using AWS EC2 Ubuntu server by Flask framework.
